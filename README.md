# Java Object-Oriented Programming (OOP)

Welcome to my personal Java Object-Oriented Programming curriculum. This repository serves as a step-by-step log of my progress as I learn the design, construction, and architectural patterns of OOP from the ground up.

---

## Curriculum Roadmap

The training is broken down into structured progressive modules, moving from raw foundations to professional software architecture.

| Module | Topic | Description |
| :--- | :--- | :--- | 
| **01** | [The Core Foundation](./Module1:TheCoreFoundation) | Classes, Objects, State Management, and IDE Terminal Workflows. | 
| **02** | Encapsulation | Data hiding, `private` bounds, and secure state validation. | 
| **03** | Inheritance | Code reusability, parent/child extensions, and structural scaling. | 
| **04** | Polymorphism | Overriding, overloading, interface execution, and dynamic routing. | 

---

##  General Environment Setup

Every module in this repository is built and tested using a standardized engineering environment:

* **Compiler/Runtime:** Java JDK 21+
* **Development Environment:** Visual Studio Code (VS Code)
* **Required Extensions:** Extension Pack for Java (by Microsoft)
